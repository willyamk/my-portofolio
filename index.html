Tech Stack

Consio adalah aplikasi pendukung keputusan yang menggunakan algoritma Weight Aggregated Sum Product Assesment (WASPAS). Dibangun dengan teknologi seperti HTML, CSS, Bootstrap, Electron.js, JavaScript, Ruby on Rails, dan MySQL. Aplikasi ini dirancang khusus untuk membantu mengambil keputusan yang lebih informasional dan beralasan.

Claudio

Tech Stack

Claudio adalah situs yang menggunakan logika fuzzy dengan algoritma Tsukamoto untuk menghasilkan keputusan yang akurat. Dibangun dengan tech stack modern seperti HTML, CSS, Tailwind CSS, Nuxt.js, TypeScript, Rust, Actix-Web, dan PostgreSQL. Dengan Rust dan Actix-Web di sisi server, Claudio menjanjikan kinerja tinggi dan keamanan yang solid.

Gotapedia

Tech Stack

Gotapedia adalah sebuah situs sistem pendukung keputusan yang menggunakan kombinasi sequence algoritma Rank Order Centroid (ROC) dan Weight Aggregated Sum Product Assesment (WASPAS) untuk analisis yang akurat. Dibangun menggunakan HTML, CSS, Tailwind CSS, jQuery, JavaScript, PHP, dan MySQL, Gotapedia menawarkan pengalaman interaktif bagi pengguna.
